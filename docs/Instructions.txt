## Problem Setup

* The search space is a **2D map**.
* Temperature values are standardized to **[0, 1]**.
* The model is allowed a fixed number of guesses.
* After each guess, the model receives a standardized temperature value.
* Axis bounds are provided and must be respected.

---

## Method

Use **Bayesian Optimization** with:

* Gaussian Process (GP)
* Squared Exponential (RBF) kernel
* Upper Confidence Bound (UCB) acquisition function

### UCB Formula

UCB(x) = μ(x) + κ · σ(x)

Where:

* μ(x) = predicted temperature
* σ(x) = model uncertainty
* κ = exploration parameter

The model must balance exploration and exploitation using κ.

---

## Acquisition Optimization (Grid Search)

Use **grid search** to select the next sampling location.

### Procedure

1. Create a uniform grid over the allowed axis bounds.
2. Evaluate the UCB value at each grid point.
3. Select the point with the highest UCB value.
4. Sample the temperature at that location.
5. Update the Gaussian Process.
6. Repeat until the allowed number of guesses is reached.

Grid resolution should provide a good balance between accuracy and speed (e.g., ~50×50).

---

## Model Behavior

At each iteration the model must:

1. Fit/update the GP using observed data.
2. Compute UCB across the grid.
3. Select the next sampling point.
4. Query temperature via `weather_api.py`.
5. Store the observation.
